# Get Your Friday - Landing Page

Premium landing page for Friday AI Executive Assistant SaaS for Thai business owners.

## Brand

- **Colors:** Deep navy (#0A0A1A), Gold (#B8963E), Cream (#EDE3D0)
- **Font:** Inter
- **Style:** Premium, luxury, minimal

## Tech Stack

- Next.js 14+ (App Router)
- React 18
- Tailwind CSS
- TypeScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Features

- ✅ Fully responsive (mobile-first)
- ✅ Smooth scroll animations
- ✅ SEO optimized
- ✅ Waitlist email collection
- ✅ FAQ accordion
- ✅ Pricing comparison
- ✅ Use case templates
- ✅ Social proof section
- ✅ Premium design with gradient effects

## Sections

1. Hero - Main value proposition with CTA
2. Problem - Pain points
3. Features - What Friday can do
4. Use Cases - Templates for different roles
5. How It Works - 3-step process
6. Pricing - 3 tiers with comparison
7. Testimonials - Social proof
8. FAQ - Common questions
9. CTA - Waitlist form
10. Footer - Links and social

## Deployment

Build for production:
```bash
npm run build
npm start
```
